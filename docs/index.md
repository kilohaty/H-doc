# HRender
[HRender](https://github.com/kilohaty/HRender) 是一个二维绘图引擎


## 引入 HRender
- npm 方式

  ```
    npm install hrender --save
  ```

- script 方式

  ```html
    <script src="yourStaticPath/H.js"></script>
  ```
